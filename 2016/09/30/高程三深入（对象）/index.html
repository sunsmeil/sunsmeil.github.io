高程三深入（对象）

发表于 2016-09-30   |   分类于 js   |  

高程三 object,function,array

array

  • 创建

  1. new操作符 创建

    var mable = new Array();

    //可以规定数组长度

    var mable = new Array(20);

    //可以初始几个值

    var mable = new Array("haha","sss","sss");

*另：new可以省略*

` var mable = Array();

//可以规定数组长度

var mable = Array(20);

//可以初始几个值

var mable = Array("haha","sss","sss");

结果相同
  1. 数组字面量表示法
//空数组
var person = []
//初始几个值
var person = ["ss","sss"]
//不要这样 ie8以下会创建3个  其他 2个
var person = [1,2,]
`

  • 访问

  1. 基于0的索引访问

    person[0]

注：可通过设置length移除最后一项

  • 检测数组

    Array.isArray(value)

  • 转换方法

    1.toString valueOf

    返回数组的字符串标示 已逗号相隔

    2.toLocalString

    返回结果相同 但调用方法不同

    3.join可以改变不同的分割符

    注:数组中任意一项值为 null or undefined ,那么以上方法返回结果空字符串

  • 栈方法（后进先出）

    1.push pop (添加 取得)

  • 队列方法（先进先出）

    1.push shift
    另有 unshift 同时使用pop可反方向模拟队列

  • 重排序

    1.reverse 反转数组顺序

    2.sort
    默认升序 （由小到大）
    比较函数 接收两个参数

    `
    function compare(v1,v2){

      if(v1 < v2){
      return -1;
  } else if(v1 > v2){
      return 1;
  }else{
      return 0;
  }
}

    `

    注：比较函数通过返回值影响排序

  • 操作方法

    1.concat

    1)添加数组 将数组放置末尾联结
    2)添加值 直接放在末尾即可

    2.slice

    1.可传 1~2个参数 1个参数 从此位置到结尾 2个参数 从第一个参数开始 到第二个参数结尾

    有负值时 用数组长度加上该值在判断 如结束小于开始空数组 如负数大于数组长度 返回所有数组

    3.splice

    1) 删除 splice(0,2)

    2) 插入 splice(2,0,”ss”,”sss”) //起始位置 要删除项数 要插入

    3) 替换 splice(2,1,”ss”)

  • 位置方法

    1. indexOf(从头查找)
    2. lastIndexOf（从末尾查找）
      找不到 返回-1 找到 返回位置索引

  • 迭代方法

    1.every 和 some
    查询数组项 满足某一条件 返回 true 或 false
    区别 every每项都是true 结果为true

    some 一项为true  结果为true

    2.filter 和map
    返回结果组成数组
    filter 指定返回的数组中 是否有某一项
    map 每项都运行传人函数 返回结果数组

    3.forEach
    没有返回值 类似for

  • 缩小方法

    1. reduce(从左向右)
    2. reduceRight(从右向左)

    `
    reduce(function(prev,cur,index,array){

    return prev + cur;

    })
    `

object

  • 创建

  1. new操作符 创建

    var person = new Object();

  2. 对象字面量表示法

    var person = { name:'ssss', age:'22' }

  • 访问

  1. 点表示法

    person.name

  2. 方括号

    person["name"]

注：常用第一种 但第二种可用于变量 属性名包含错误语法字符时

  • 应用环境（方括号）

    相同类型数据提交

function

函数名是指针 函数是对象

  • 定义

    1. function asd (n1,n2){}
    2. var s = function(n1,n2)
    3. var s = new Function ("n1","n2","return n1+n2")//不推荐

  • 没有重载

  • 函数声明与函数表达式

    函数声明 可以在它之前调用 函数表达式不可以
    //没有问题 alert(sun); function sun(){ return 1; } //报错 alert(sun); var sun = function(){ return 1; }

  • 内部属性

    1. arguments(保存函数参数)
      arguments.callee 指向拥有这个arguments对象的函数
    2. this

  • 属性 方法

    1. length 接收命名参数的个数
    2. prototype 保存所有实例方法
    3. apply 和 call 区别在于call明确传递每一个参数 可用来扩充作用域
